🎯 KEY INSIGHTS

The RAG system is a sophisticated 10-stage pipeline:

  Query → Expand (1-4 variants)
  → Semantic Search + Keyword Search (parallel)
  → Merge & Filter (similarity thresholds)
  → Neighborhood Expansion (context)
  → Reference Expansion (document links)
  → Rerank (heuristic + optional ML)
  → Compress (block scoring)
  → Format → Output

Critical thresholds:
  • Min similarity: 0.28 (confidence floor)
  • Similarity window: 0.12 (dynamic range)
  • Max context: 20,000 characters
  • Max sections: 12 total

Multi-turn features:
  • Query rewriting (LLM decides SKIP vs RETRIEVE)
  • Context reuse (avoid redundant searches)
  • Recent document boosting (+15 score)

Graceful fallbacks:
  • No embeddings? → Keyword-only search
  • Reranker down? → Heuristic ranking
  • Reference expansion fails? → Continue anyway
